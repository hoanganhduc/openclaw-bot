# Intent Interview — shapes

## Per-question shape

    Q: <one focused question>
    GUESS: <your current hypothesis + the reasoning behind it>

## Confirmed-intent restate (six lines)

    Outcome:      <what exists / is true when this is done>
    User:         <who the result is for>
    Why now:      <the trigger or motivation>
    Success:      <the concrete, checkable bar>
    Constraint:   <the binding limit — time, tool, scope, policy>
    Out-of-scope: <what this explicitly does NOT cover>

## Status values

- `CONFIRMED` — the user gave an explicit yes to the restate
- `NEEDS-INPUT` — still below confidence; more questions pending
- `BLOCKED-NON-INTERACTIVE` — no user available; proceeding on stated assumptions
