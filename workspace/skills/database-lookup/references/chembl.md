# ChEMBL

- Base URL: `https://www.ebi.ac.uk/chembl/api/data`
- Best for: molecule, target, activity, mechanism-of-action, and drug-indication records
- No API key required
- Core endpoints: `/molecule`, `/target`, `/activity`, `/mechanism`, `/drug_indication`
- Useful when you need assay or potency context around a compound or target
