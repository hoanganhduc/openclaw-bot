# PubChem

- Base URL: `https://pubchem.ncbi.nlm.nih.gov/rest/pug`
- Best for: compound, substance, and assay lookups by name, CID, SMILES, InChI, or InChIKey
- No API key required
- Useful operations: `record`, `property`, `synonyms`, `description`, `cids`, `xrefs`
- Good first step when you need canonical identifiers or basic chemical properties
